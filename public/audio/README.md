# Template Music Files

Place MP3 files here with these exact names:

- `maharani.mp3` — Indian wedding instrumental (sitar, tabla, shehnai)
- `azure-vows.mp3` — Soft acoustic/ocean ambient
- `whispered-vows.mp3` — Vintage/nostalgic piano
- `noir-atelier.mp3` — Corporate ambient/jazz
- `maison-blanche.mp3` — Elegant piano/classical

## Where to find free music:
- https://pixabay.com/music/ (search "wedding", "ambient", "piano")
- https://www.bensound.com/
- https://freemusicarchive.org/

Download the MP3 and rename it to match the filename above.
If no MP3 is found, the app will use a Web Audio API ambient tone as fallback.
